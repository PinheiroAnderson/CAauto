# CA Auto Serviços

Site institucional para a oficina CA Auto Serviços, especializada em mecânica automotiva em Guaratiba, RJ.

## Descrição

Este projeto é um site responsivo para apresentar os serviços da oficina CA Auto Serviços, incluindo freios, injeção, suspensão, óleo e bateria. O site foi desenvolvido utilizando HTML, CSS, JavaScript e Vue.js para uma experiência interativa e moderna.

## Funcionalidades

- **Design Responsivo**: Layout adaptável para desktop e mobile
- **Saudação Dinâmica**: Mensagem de boas-vindas que muda conforme o horário do dia
- **Navegação Suave**: Scroll suave para seções da página
- **Menu Hambúrguer**: Navegação mobile com acessibilidade
- **Componentes Vue.js**: Seção de serviços utilizando componentes reativos
- **Ano Atual Automático**: Exibição dinâmica do ano no rodapé

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da página
- **CSS3**: Estilização com variáveis CSS e design responsivo
- **JavaScript (ES6+)**: Interatividade e lógica da aplicação
- **Vue.js 3**: Framework para componentes reativos
- **Font Awesome**: Ícones vetoriais
- **Google Fonts**: Fonte Montserrat

## Desenvolvimento

O projeto utiliza Vue.js via CDN para simplicidade. Para desenvolvimento avançado, considere migrar para Vue CLI ou Vite.

### Scripts

- `script.js`: Contém a aplicação Vue.js principal com:
  - Dados reativos para saudação e serviços
  - Componente `service-item` para exibição dos serviços
  - Métodos para atualização dinâmica e interatividade
